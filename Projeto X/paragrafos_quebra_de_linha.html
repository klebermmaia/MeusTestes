Paragrafos e quebras de linha

Paragrafos

para escrever um parágrafo basta usar a Tag <p> e </p>, todo o texto deve ir entre a Tag.
Como visto no Exemplo abaixo.

Tag: <p> Exemplo de Paragrafo. </p>.

OBS.: Paragrafos é algo semantico para o HTML5, no caso dessa Tag não é algo visivel, é apenas para compreenção dos navegadores. Mas cada Paragrafo (Tag <p>) que você usar vai ser colocado em uma nova linha, eliminando a necessidade de uma Tag <br>

Quebra de linhas

  Para quebrar linhas basta usar no final da linha ou palavra que deseja que seja quebrada usa-se a Tag <br>. A Tag <br> serve para quebrar as linhas quantas vezes quiser, basta usar várias vezes em sequência, como no exemplo abaixo.

   Aqui um exemplo de <br>
---
quebra de quebra de linha.
  Ou você pode quebrar mais do que <br> <br>
---
---
uma linha.
