HTML5, CSS3, JS e muito mais.

html

HTML é uma sigla que vem do ingles que significa Hyper Text Markup Language, ou no porquetes Hiper Linguagem de Marcação de Texto. O HTML tem como objetivo inserir conteudo e estabelecer estruturas para um Website.

css

CSS é uma singla que vem do ingles que siginifica Cascading Style Sheets, ou no portugues Folha de Estilo em Cascatas. O CSS tem como objetivo trazer estilo ao um Website.

js

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web, toda vez que uma página da web faz mais do que simplesmente mostrar a você informação estática, mostrando conteúdo que se atualiza em um intervalo de tempo, mapas interativos ou gráficos 2D/3D animados, etc
