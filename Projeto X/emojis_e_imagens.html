Emojis, Símbolos e Imagens

Emojis

Como adicionar emojis? primeiro Você deve ir ao site emojipedia.org, entrando no site escolha o Emoji desejado, copie o Codepoints do Emoji escolhido (menos o U+), vá ao seu editor e digite onde deseja que fique, digite: &#x + o Codepoints (tudo junto) e pronto.

  Exemplo: &#x + o Codepoints 1F596 = &#x1F596 (🖖)

Adicionando Imagens

Para adicionar primeiro escolha a imagem que deseja colocar em seu site, após escolher salve em uma nova pasta do seu projeto (uma pasta dentro da pasta onde se localiza o index. Para manter organizado). No seu editor escolha onde a imagem vai e escreva a Tag: <img src=" " alt= " ">. O preenchimento da Tag é dessa maneira: src é o local de origem, como estamos usando uma pasta no mesmo local do arquivo index basta por o nome da pasta e o nome do arquivo. Exemplo: <img src="imagem/imagem.png" (OBS.: a extensão do arquivo é muito importante). Em alt=" " é o texto que melhor define essa imagems, não é obrigatório ser preenchido.
Para carregar uma imagem externa (uma imagem externa seria uma imagem da web) basta ter o link da imagem que você ira user e no src basta por entre as aspas duplas, exemplo: src="Link da imagem"

como adaptar imagens de tamanho diferentes

para adaptar uma imagems para varios tamanha de tela podemos utilizar a tag <picture> e dentro dela mais duas, img e source.
ficara dessa maneira:

<picture>
<source media="(max-min-width: (aqaui o tamanho maximo da imagem))" srcset="(a imagem que vai ser colocada de acordo com o tamanho)" type="image/(aqui depos da barra o pré-formatado da imagem)">
<source media="(max-min-width: )" srcset="" type="image/">
<img src="(imagem inicial)" alt="">
</picture>

a ordem é muito importante entre os source, se a imagem iniciar for grande logo a do meio é sempre a media e o primeiro source é a imagem pequena.
