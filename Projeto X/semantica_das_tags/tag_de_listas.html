Listas

a tag OL cria uma lista ordenada e a UL cria uma lista não ordenada, já a tag LI cria os itens dentro dessas tags.
Exemplo:
    <ol>
    <LI> item 1.</li>
    <LI> item 2.</li>
    <LI> item 3.</li>
    <LI> item 4.</li>
    <li> item 5.</li>
    </ol>
 se escreve dessa maeira tanto a OL quando Ul. Mas ficaria desse jeito.
  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5
note que com a OL existe numeros antes de cada item, isso se deve por se uma lista ordenada. Isso pode ser alterado da seguinte forma, dentro da tag <ol> vá deppois de ol e digite o parametro type="" dentro de type voce tera algumas opções são elas 1, A, a e I.
1: é uma lista numerica.
A: é uma lista alfabetica de letras maiusculas.
a: é uma lista alfabetica de letras menusculas.
I: é uma lista de numeros romanos.
Tambem tem o parametro start="" que se você usar o type 1 e quiaser que sua lista comece com 2 basta colocar 2 em start.
exemplo:
    <ol type="1" start="2">
    <LI> 2. item 1.</li>
    <LI> 3. item 2.</li>
    <LI> 4. item 3.</li>
    <LI> 5. item 4.</li>
    <li> 6. item 5.</li>
    </ol>
 o codigo ia facar da seguinte maneira
  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5
isso vale para qualquer type em romano por exemplo.
  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5
