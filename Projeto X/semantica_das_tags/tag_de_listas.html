Listas

a tag OL cria uma lista ordenada e a UL cria uma lista não ordenada, já a tag LI cria os itens dentro dessas tags.
Exemplo:
    <ol>
    <LI> item 1.</li>
    <LI> item 2.</li>
    <LI> item 3.</li>
    <LI> item 4.</li>
    <li> item 5.</li>
    </ol>
 se escreve a tag <ol> dessa maneira ficaria desse jeito.
  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5
note que com a OL existe numeros antes de cada item, isso se deve por ser uma lista ordenada. Isso pode ser alterado da seguinte forma, dentro da tag <ol> e digite o parametro type="", em type você tera algumas opções são elas 1 (que é a padrão), A, a e I.
1: é uma lista numerica.
A: é uma lista alfabetica de letras maiusculas.
a: é uma lista alfabetica de letras menusculas.
I: é uma lista de numeros romanos.
Tambem tem o parametro start="" que se você usar o type 1 e quiaser que sua lista comece com 2 basta colocar 2 em start.
exemplo:
    <ol type="1" start="2">
    <LI> 2. item 1.</li>
    <LI> 3. item 2.</li>
    <LI> 4. item 3.</li>
    <LI> 5. item 4.</li>
    <li> 6. item 5.</li>
    </ol>
 o codigo ia facar da seguinte maneira
  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5
isso vale para qualquer type até em romano, por exemplo:
  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5
listas não ordenadas é inserido uma decoração diferente. No type você pode mudar essa decoração, são elas disc (que é a padrão), circle, square.
exemplo:

disc

  • disc

circle

  • circle

square

  • square

listas mistas

é possivel mescla as lista, colocando uma ul dentro de uma ol ou visse versa.
exemplo:
  1. exemplo 1
    • exemplo 1.1
  2. exemplo 2
    • exemplo 2.1
ficaria dessa mandeira:
    <ol>
    <li>exemplo 1</li>
    <ul><li>exemplo 1.1</li></ul>
    <li>exemplo 2</li>
    <ul><li>exemplo 2.1</li></ul>
    </ol>

Bonus

tag dl cria uma lista de definiçõa, tag dl para criar a lista, tag dt para criar o termo e tag dd para criar a definiçõa.
Exemplo:
exemplo
exemplo é algo para fazer uma mostragem de algo para compreenção
se escrever dessa maneira
<dl>
<dt>exemplo</dt>
<dd>exemplo é algo para fazer uma mostragem de algo para compreenção</dd>
</dl>
