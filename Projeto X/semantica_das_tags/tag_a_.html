Ancora

O elemento <a> em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a> precisará indicar o destino do link.
exemplo de codigo:
este é um exemplo que manda o usuario para pagina inicial
codigo: este é um exemplo que manda o usuario para <a href="../index.html">pagina inicial</a>
por padrão quando se click em um link ele muda a pagina atual da sua aba, mas para que isso não ocorra pode se usar o parametro target="" e atribuir _blank para que abra em uma nova aba.
