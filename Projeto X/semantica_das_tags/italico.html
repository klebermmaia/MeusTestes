Tag <I> ou <EM>?

A tag <I> é apenas para deixa visualmente em itálico (não semântica), já a tag <EM> tem semântica trazendo sentido de enfase a palavra ou frase em itálico, assim os navegadores e buscadores saberão que a um texto marcado como itálico na página. caso se use a tag <I> irá ser algo identificado pelo usuario e nada mais.
