Tags

  • <H1 a H6> -> usado para Titulos. (Exemplos)
  • <P> -> usado para criar paragrafos.
  • <BR> -> Usado para quebrar a linha.
  • <LINK> -> Usado para criar uma ligação entre arquivos.
  • <A> -> Usado para criar um ligação de forma mais variada. (Exemplo)
  • <IMG> -> usado para adicionar imagens.
  • <Strong> e <B> -> A tag <B> deixa o texto marcado em negrito e nada mais, porem a tag <Strongb> tem semantica, ela da sentido de texto em destaque. (Exemplo)
  • <I> e <EM> -> Deixa em Italico. (Exemplo)
  • <MARK> -> Deixa o texto marcado (igual a caneta marca texto).( Exemplo )
  • <SMALL> -> Deixa o texto pequeno.
  • <INS> -> Deixa o texto com uma linha embaixo para você considerar importante.
  • <SUP> -> Usado para fazer expreções de potencia, por exemplo: 102= 100
  • <SUB> -> Usado para fazer expreçoes de formulas, por exemplo: H2O
  • <Ul>,<OL> e <LI> -> Cria lista. Exemplo
  • <CODE> -> Marca o texto como codigo.
  • <PRE> -> Permite que seja considerado tudo que estaja entre a tag, deixa o texto pré-formatado.
  • <blockquote> -> Usado para fazer uma citação em bloco com deslocamneto lateral. (Exemplo)
  • <ABBR> -> Usado para abreviações. Quando passa o mouse em cima da palvra mostra a frase da abreviação. (Exemplo )
