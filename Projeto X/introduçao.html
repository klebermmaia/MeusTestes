HTML

Vamos aprender

O que é HTML? Simples, HyperText Markup Language, Significa Linguagem de marcação de hipertexto. Digamos que seja um jeito de editar paginas de sites para Navegadores podendo fazer formatação de texto. Essa ferramenta é usada para que possamos editar o nosso conteudo. Para que possamos dar estilo/Beleza ao nosso site é necessário o uso do CSS para estilizar o site.
O HTML em versões anteriores a 5º versão era apenas marcaçoes de texto para que possamos editar o nosso conteudo, mas com a chegada da 5º versão foi dado sentido as Tags.
Isso é, um texto em negrito é usado para dar Destaque, mas o que muda? simples a imterpretação, que é voltada para os navegadores saberem que aquele texto em questão esta destacados, para nós que vemos é claro o destaque.
  Por exemplo a Tag <header></header> não tem mudança visivel porem ela está la indicando que tudo que está entre ela é um cabeçalho.

A importancia da semântica

Semântica é o significado, num sistema linguístico, o componente do sentido das palavras e da interpretação das sentenças e dos enunciados.
No HTML é incide sobre a relação entre significantes, tais como palavras, frases, sinais e símbolos, e o que eles representam, a sua denotação. A semântica linguística estuda o significado usado por seres humanos para se expressar através da linguagem.

Vamos iniciar um documento HTML

Vamos usar nesse tutorial o Visual Studio Code (VS Code). Primeiro vá para a pasta documentos em seu PC (ou em outra pasta desejada) e crie uma nova pasta para menter tudo organizado, a nova pasta pode se chamar 'Teste' dentro dela crie uma nova pasta com o nome HTML.


  Agora no VS Code vamos abrir a pasta, vá em Arquivo, Abrir Pasta.

Selecione a pasta html e a cima tera quatro icones o primeiro é para criar um novo arquivo, clique nesse e em seguida nomeio com o nome index.html (note que esta em menusculo, escreva sempre os nomes dos arquivos em menusculo), note que é importante colocar a extenção do arquivo que no caso é .html, no VS code existe outras linguagens de programação você deve ser bem especifico com o que você quer.

Vai se abrir a pagina em branco, digite nela ! e aberte enter ou clica na opçõe que aparece ao digitar a exclamação ou você pode digitar html:5, ira aparecer o seguinte:


▸Na primeira linha vemos: <!DOCTYPE html> que declara que é um documento tipo HTML.
▸Segunda linha: <html lang="en"> diz ao navegador que seu site qual o idioma, vem em ingles mas pode mudar colocando entra as "pt-br".
▸Terceira linha: <head> aqui temos a cabeça, note que existe outra na linha sete, é a mesma Tag porem esse é o fechamento. algumas Tags possuem fechamento e outras não, a maioria possui o fechamento, mas não se preocupe em saber porque tem ou não tem, isso é uma questao muito além do que estamos vendo aqui. <head> é onde se faz algumas definições do site e onde podemos colocar link para puxar o css e o JavaScrip, vamos ver mais a frente.
▸Quarta linha: <meta charset="UTF-8"> aqui você não mexe. Essa é uma configuração para nosso alfabeto, que contem acentuação.
▸Quinta linha: <meta name="viewport" content="width=device-width, initial-scale=1.0"> é uma predefinição da sua pagina, não se muda.
▸Sexta linha: <title> simples é apenas um titulo onde você pode ver nos indicadores de aba dos navegadores, coloque um titulo do seu desejo.
  Agora vamos ver o <body> onde vai tudo que você quer que o usuário veja como texto, imagem, video e etc.
