como adicionar degrade

para criar um degrader no seu site basta criar um style e usar o comando background-image: linear-gradient(); o parametro linear-gradient() criar um degrade basta por entras os () o sentido ou angulo por exemplo to bottom (para baixo) ou 45deg (45° graus).
exemplo: um style local

<style>
body{
background-image: linear-gradient(150deg, #F3FEB0, #ff8400);
background-attachment: fixed;
}
</style>
