Tags

Tags são formadas por < > e dentro vai uma tag, como por exemplo <h1> essa é uma Tag de Titulo que vai de h1 a h6. Tags Em sua maioria começam com <Tag> e terminha com </Tag>

Tags Basicas no html5

Para iniciar um documento em html5 pelo VS Code, você pode escrever uma interrogção (!) e aperta enter que vai gerar essas tags automaticamente. Na segunda linha onde tem lang="en", você pode mudar para pt-br assim os navegadores vão saber que sua pagina esta em portugues. A tag head é a 'cabeça' onde vai configurações da pagina (linua, padrao do alfabeto, estilos, links e etc), já a tag body vai o conteudo que vai ser visto pelo usuario
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document/title>
</head>
<body>
(Coteudo)
</body>
</html>

H1 a H6, Paragrafos e Quebra de linhas

Tag P ou Paragrafos

A tag <P> é a tag para criar paragrafos, ou seja quando voce quise declara que voce esta iniciando um paragrafo basta começar com a tag <P>, escreva seu texto e termine com </P>

Quebra de Linhas

A quebra de linha é feito com a tag <br>, é uma tag que não tem fechamente isso é não precisa usar </br> pora finalizar uma quebra de linha, basta por <br>. Porem versoes antigas do html se usava </br> enves de <br>.

Links

Um link é uma ligação entra arquivos ou paginas, no caso de um arquivo se usa a tag <link>, usada dentro da tag <head> fica dessa maneria <link rel="stylesheet" href="style/style.css"> (nesse caso é um link para fazer uma ligação entro o arquivo html e o css). Com a tag <a> voce pode usar dentro da tag <body>, lhe permita navegar entre:
interno: redireciona para um elemento existente dentro da mesma página;
local: utilizados para páginas contendo o mesmo domínio, ou seja, entre páginas do mesmo site;
global: utilizados para páginas de outros domínios, ou seja, fora do site.

Adicionando imagens

Para adicionar imagens voce pode usar a tag <img> ela vai ficar assim <img src="" alt="">, src é o local onde esta a imagen e alt é o nome alternativo que vai aparecer caso não carregue a imagem, salve em uma pasta onde tem seu arquivo index, então voce digita o nome da pasta que voce criou e dps da barra o nome da imagem com a extenção do arquivo (.png, .jpg e etc) como por exemplo src="imagens/acessibilidade-web.png",
